# Image Retrieval
Implementation of a CBIR in C++

## How to Run
To compile use:
    
    cmake .

And to run:
    
    ./image_retrieval

Then:\
     1) For color histogram distance\
     2) For shape distance\
     3) For ORB descriptors distance\
     4) For SIFT descriptors distance\
     5) For ORB + SIFT descriptors
